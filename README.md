# 📊 Analisis Data Produksi Sampah (Python & Pandas)

Repositori ini berisi proyek pemrosesan dan analisis data menggunakan **Python** dan **Pandas DataFrame**. Proyek ini difokuskan pada pengolahan data mentah terkait volume produksi sampah menjadi laporan statistik yang terstruktur dan mudah dipahami.

## 🗂️ Deskripsi Dataset & Output
Sistem memproses data mentah awal (`DataSampah.csv`) dan menghasilkan beberapa laporan agregasi akhir dalam format `.csv` dan `.xlsx`, yang meliputi:
*   **Hasil Produksi Sampah:** Rekapitulasi keseluruhan produksi.
*   **Total Produksi Per Kota & Tahun:** Analisis distribusi geografis dan tren deret waktu (*time-series*).
*   **Total Produksi Per Tahun:** Pemantauan tren kenaikan/penurunan volume sampah tahunan.

## 🛠️ Teknologi & Tools
*   **Bahasa Pemrograman:** Python
*   **Library Utama:** Pandas (Untuk *Data Manipulation & Analysis*)
*   **Environment:** Jupyter Notebook (`.ipynb`)
*   **Format Ekspor:** CSV & Microsoft Excel

## 🚀 Cara Menjalankan
1. Pastikan Anda telah menginstal Python beserta library Pandas.
2. Clone repositori ini ke dalam direktori lokal Anda.
3. Buka file `.ipynb` menggunakan Jupyter Notebook, VS Code, atau Google Colab.
4. Jalankan sel kode (*cells*) secara berurutan untuk melihat proses impor data, pembersihan data (*cleaning*), agregasi (*grouping*), hingga ekspor data ke format Excel/CSV.
